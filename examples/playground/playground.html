Here you can configure a chart by changing the values in the form on the left side of the page. Only use this in FF or Chrome, too slow/untested otherwise.

After every change, the code to plot the chart is updated.
The checkboxes on the left indicate if you use the custom value (checked) or the default value (unchecked) of the options.
To change the options of a collapsed group, you must click on its title.

Please, report any bugs here.
